# TaskJET - Task Management Website

TaskJET is a task management website designed for students to help organize their tasks efficiently.

## Live Demo
[TaskJET Live Demo](https://taskjet-36f09.web.app/)


## server side code: (https://github.com/Mehedimid/Task-Jet-Server)

## Technologies Used
- Frontend:
  - React
  - Tailwind CSS
  - Firebase (Authentication)
- Backend:
  - Node.js
  - Express.js
  - MongoDB

## Features
- User authentication with Firebase
- Responsive design for both desktop and mobile devices
- Task management with options to add, edit, and mark tasks as ongoing or completed
- ...

## Getting Started
### Prerequisites
- Node.js
- MongoDB
